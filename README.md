# Ismail El Jamiy — Personal Portfolio

Personal portfolio website for **Ismail El Jamiy**, PhD candidate in AI at the University of Toulouse (IRIT / IMT / CLLE).

Research focus: Handwritten Text Recognition of medieval Arabic scientific manuscripts and dedicated Large Language Models.

## Stack

- [Astro 4.x](https://astro.build) — static site framework
- [Tailwind CSS](https://tailwindcss.com) — utility-first styling
- [MDX](https://mdxjs.com) — content with components
- Hosted on [Vercel](https://vercel.com)

## Languages

English (default) · French (`/fr/`) · Arabic (`/ar/`, RTL)
